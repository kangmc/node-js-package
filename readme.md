## Name : javascript package based on Node.js
## Version : 1.0

이 프로젝트는 Node.js에 의해 빌드되며, Express, EJS를 사용하여 템플릿을 구성하였습니다.

### 사용방법

1. node 설치
[Node](http://nodejs.org/) 사이트에서 플랫폼에 맞는 버전 설치

1. Git 설치

1. git에서 소스코드 내려받기
git clone https://https://github.com/6feetsong/node-js-package

1. Node 패키지 설치
npm install

1. Node 패키지 업데이트
npm update
