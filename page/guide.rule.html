Naming Rule

Camel Case : 변수, 함수명, 인스턴스
Pascal Case : Class 명
Upper Case :
$ : jQuery 롤 Select 한 객체는 prefix 로 '$' 를 사용한다. 
			// 변수
			var str, i, arr;
			
			// 함수
			function func () {}
			
			// 인스턴스
			var now = new Date();
		
			// 클래스 명
			function ClassName (){
				
			}
		
			// jQuery DOM
			var $target = $('#targetId');

Code Rule

- 가급적 전역으로 설정되는 변수, 함수를 사용하지 않는다.
- 단일 var : 변수의 선언시 사용되는 var 는 연관영역 안에서 한번만 사용한다.
- 변수 선언시 가급적 초기값을 지정한다.
- for 문 작성시 length 를 저장하여 사용한다. 
			// 전역 침범하지 않기
			(function() {
				// blah, blah...
			})();
			
			// 단일 var
			var var1, var2, arr1, arr2;
			
			// 초기값 지정
			var str1 = '',
				str2 = '',
				num1 = 1,
				arr1 = [];
			
			// for 문 length 저장
			var i = 0, len = 0;
			for(i = 0, len = 10, i < 10; i ++) {
				// some...
			}
