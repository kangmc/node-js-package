플로그인은 프로젝트단위로 packaging 하여 사용한다.
플러그인 제작시 한개 파일은 50 KB 를 넘지 않도록 한다.

1. js 파일 등록

아래의 내용을 </head> Tag 바로 위에 작성한다. 
			<script src="./scripts/jquery/jquery-1.10.2.min.js"></script>				
		    <script src="./scripts/jquery/plugin/jquery.plugin.pkg.js"></script>		
			<script src="./scripts/jrx/jrx.min.js"></script>						
			
			<script src="./scripts/init.js"></script>

2. Initialize

동일하게 적용되어야하는 script는 scripts/init.js 파일에 작성한다.
application configuration, ajax 설정 정보 등을 여기서 작성한다.

Example

			(function( $, window, document){
				$(function(){
					
					//================== application configuration ===============
					var _appconfig = {
							"useLog" : true
						};
					// set application configuration.
					$.config(_appconfig);
					
					//...        
					//============================================================
					
					//...
					
					//===================== global initialize ====================
					// {Here!!!}
					//============================================================
					
					//...
					
				});
			})( jQuery, window, document );

3. JavaScript Code Block (Page)

page 에 작성되어야 하는 script 는 </body> Tag 바로 위에 작성한다.

Example

			<script>
			(function( $, window, document){
				$(function(){
					
					//======================== page initialize ===================
					// {Here!!!}
					//============================================================
					
				});
			})( jQuery, window, document );
			</script>
			</body>

4. Folder Structure

Scripts 폴더의 구조를 설명한다.

(d) scripts : 모든 scirpt 파일을 저장한다.
(d) scripts > lib : jquery 를 제외한 모든 scirpt 파일을 저장한다.
(d) scripts > jquery : jquery 관련된 모든 파일을 저장한다.
(d) scripts > jquery > plugin : jquery plugin 파일을 저장한다.
			scripts/

5. Rule

Camel Case, Pascal Case.

5-1. Naming Rule

			// variable : Camel Notation.
			var 
			
			// class : Pascal Notation.
			function ClassName (){
				
			}
			
			// function : Camel Notation.
			function func () {}

5-2. Plugin Rule

- 프로젝트별 plugin.package 를 작성하여 사용한다.
- 모든 플러그 인은 minify 상태로 사용한다.

5-3. Code Rule

- 가급적 전역으로 설정되는 변수, 함수를 사용하지 않는다.
- 단일 var : 변수의 선언시 사용되는 var 는 연관영역 안에서 한번만 사용한다.
- 변수 선언시 가급적 초기값을 지정한다.
- for 문 작성시 length 를 저장하여 사용한다. 
			// 전역 침범하지 않기
			(function() {
				// blah, blah...
			})();
			
			// 단일 var
			var var1, var2, arr1, arr2;
			
			// 초기값 지정
			var str1 = '',
				str2 = '',
				num1 = 1,
				arr1 = [];
			
			// for 문 length 저장
			var i = 0, len = 0;
			for(i = 0, len = 10, i < 10; i ++) {
				// some...
			}
