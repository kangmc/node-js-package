DOM Select 횟수 줄이기

							동일한  Selector 가 두번이상 나는 경우,
							처음 Selected 한 정보를 사용한다.

Bad Case

							function someFunction(){
								
								$('#contents').height(100);
								$('#contents').append('
some string
');
								$('#contents').find('.item').remove();
							}

Good Case

							function someFunction(){
								
								var $contents = $('#contents');
								
								$contents.height(100);
								$contents.append('
some string
');
								$contents.find('.item').remove();
							}

.find() 를 사용해라

Bad Case

							$('#contents a')

Good Case

							$('#contents').find('a')
