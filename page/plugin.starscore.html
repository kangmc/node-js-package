jQuery Plugin : starScore

별점 주기 기능
selector target 의 data-target Property 설정으로 설정된 값을 INPUT 에 넣을 수 있다.
* selector target : 별점 Element 를 감싸고 있는 SPAN.

			
			<span class="star-score-check-19h" data-score="0" data-target="#point_price">
				<button type="button" value="5" class="even">5점</button>
				<button type="button" value="4.5">4.5점</button>
				<button type="button" value="4" class="even">4점</button>
				<button type="button" value="3.5">3.5점</button>
				<button type="button" value="3" class="even">3점</button>
				<button type="button" value="2.5">2.5점</button>
				<button type="button" value="2" class="even">2점</button>
				<button type="button" value="1.5">1.5점</button>
				<button type="button" value="1" class="even">1점</button>
				<button type="button" value="0.5">0.5점</button>
			</span>
			<input type="hidden" name="point_price" id="point_price" value="0" />
			
			<script>
				$('span[class^="star-score-check"]', $doc).starScore();
		    </script>
