jQuery Plugin : listItem

상품아이템에 대한 Interaction Control.
장바구니 레이어 show/hide : 해당 아이템영역(.item) 에 div.layer-option 요소가 있는 경우 동작.
수량 증가/감소

* selector target : 상품리스트 UL

* options
- detail {Function} : 상세 아이콘 클릭 이벤트
- zzim {Function} : 찜하기 클릭 이벤트
- cart {Function} : 장바구니 클릭 이벤트 (장바구니레이어 있는 경우, 레이어의 장바구니 버튼)
	    	$('#goods_list').listItem({
	        	detail : function () {},
	        	zzim : function () {},
	        	cart : function () {
	        		
	        		$.log('cart');
	        	}
	        });
