jQuery Plugin : listAccordion

UL, Table 태그를 대상으로 제목 클릭시, 내용 부분을 show 시킨다.

* selector target : UL, Table

* options
- type {String} : default is 'one', 'all' 
			<ul id="review_list">
				<li>
					<h3><a href="#">제목1<span title="조회수">15</span></a></h3>
					<div class="answer" id="faqList1">
						<p>내용1</p>
					</div>
				</li>
				<li>
					<h3><a href="#">제목2<span title="조회수">9854</span></a></h3>
					<div class="answer" id="faqList2">
						<p>내용2</p>
					</div>
				</li>
			</ul>`
			
			<script>
				// 한개의 row 만 show.
		    	$('#review_list').listAccordion();
				
				// show 상태 유지.
				$('#review_list').listAccordion({
					type : 'all'
				});
		    </script>
