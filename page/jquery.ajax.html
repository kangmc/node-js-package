AJAX

jQuery.ajax 에 대한 설정 관리와
get, set, formsubmit, jsonp 사용 방법에 대해 알아본다.

참조 : jQuery.ajax()

기타 참조 : .ajaxError(), jQuery.ajaxPrefilter(), jQuery.ajaxSetup()

get/set

			// get
			var url = 'http://yourdomain.com/api';
			$.ajax({
				url : url
			}).done(function (r) {
				// 성공 후, callback
			});
			
			// set
			var url = 'http://yourdomain.com/api';
			$.ajax({
				url : url,
				type : 'POST'
			}).done(function (r) {
				// 성공 후, callback
			});

jsonp

jsonp 방식을 사용하려면 아래 코드처럼 dataType 을 'jsonp'로 셋팅하면 된다. 
			// set
			var url = '...';
			$.ajax({
				url : url,
				dateType : 'jsonp',
			}).done(function (r) {
				// 성공 후, callback
			});

전역 설정

jQuery 에서 jQuery.ajax 에 대해 기본값을 설정할 수 있는 ajaxSetup method 를 제공한다.
자세한 스텍은 아래 참조 링크에서 확인 가능하며,
본 Package 에서는 scripts/jquery/jquery.init.js 파일에 해당 코드가 존제한다.

참조 : jQuery.ajaxSetup() 
			$.ajaxSetup({ 
				cache: false,
				converters: {
					// Convert anything to text
					"* text": window.String,
					// Text to html (true = no transformation)
					"text html": true,
					// Evaluate text as a json expression
					"text json": jQuery.parseJSON,
					// Parse text as xml
					"text xml": jQuery.parseXML
				}
			});
