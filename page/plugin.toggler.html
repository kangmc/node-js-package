jQuery Plugin : Toggler

버튼 클릭시 특정 영역을 toggler.
trigger button click > toggle_area toggling

* options
- target {jQuery Selector} : toggle target
- close {jQuery Selector} : target 내부에 닫기 버튼이 존재하는 경우 사용.
- toggleClass {String} : 기본동작은 display:none/block 이지만 class 값으로 toggle 되도록 변경이 필요한 경우 사용.
- isDocTrigger {Boolean} : target 이외의 document 영역을 클릭시 자동으로 닫히게 하는 설정.
- toggle {Function} : toggle 될 때마다 항상 동작하는 함수.
- beforeShow {Function} : show 전에 동작하는 함수.
- afterHide {Function} : hide 후에 동작하는 함수. 
			<button type="button" id="btn_toggle" title="Toggle">Toggle</button>
			<div id="toggle_area"></div>
			
			<script>
		    	$('#btn_toggle').toggler({
		        	target : '#toggle_area'
		        });
		    </script>
