jQuery Plugin : priceSlider

가격 정보 슬라이더 핸들러.

* options
- min {jQuery DOM} : min 값 표기 영역. SPAN
- max {jQuery DOM} : max 값 표기 영역. SPAN
- start {jQuery DOM} : start 값 표기 영역. INPUT
- end {jQuery DOM} : end 값 표기 영역. INPUT
			<div class="input-price-wrap">
				<input type="tel" id="search_price_start" title="최소 가격 입력하기" value="5000"/> 원 ~ <input type="tel" id="search_price_end" title="최대 가격 입력하기" value="70000"/> 원
			</div>

			<div class="slider-price-wrap" id="search_price_range">
				<span class="view-price-min" id="search_price_min">2,000</span>
				<span class="view-price-max" id="search_price_max">1,000,000</span>
				<!-- //가격 제어 -->
				<div class="price-bar-default"></div>
			</div>
			
			<script>
	    	$('#search_price_range').priceSlider({
		    	min : $('#search_price_min'),
		    	max : $('#search_price_max'),
		    	start : $('#search_price_start'),
		    	end : $('#search_price_end')
		    });
		    </script>
