Packaging

- Plugin 파일은 프로젝트 단위로 concatenate 된 minify 버전을 사용한다.

- Build Environment
: grunt.
  • Node 설치
  • grunt cli 설치
  • package.json, gruntfile.js 추가
  • plugin 설치
  • minify 실행

Node 설치

grunt 를 실행할 node 환경 설치를 위해 아래 사이트에서 node를 설치한다.
(http://www.nodejs.org/)

Command Prompt 에서 npm -v 입력시 현채 설치된 버전정보가 나타나면 정상적으로 설치된 것이다. 
			$ npm -v
			
			$ 1.4.32

grunt cli 설치

			$ npm install -g grunt-cli

package.json, gruntfile.js 추가

아래 두 파일을 다운받고 프로젝트 Root 에 추가한다.

package.json Download
gruntfile.js Download

gruntfile.js 파일에서 packaging 할 구조를 설정한다.

			jquery_plugins: {
				src: [
					'<%= pkg.path.plugin %>/jquery.navigate.js',
					'<%= pkg.path.plugin %>/jquery.stateAlarm.js',
					'<%= pkg.path.plugin %>/jquery.countdown.custom.js', 
					'<%= pkg.path.plugin %>/jquery.layer.js', 
					'<%= pkg.path.plugin %>/jquery.number.adjust.js',
					'<%= pkg.path.plugin %>/jquery.rolling.js',
					'<%= pkg.path.plugin %>/jquery.timer.js',
					'<%= pkg.path.jquery %>/jquery-migrate-1.2.1.js'
				],
				dest: '<%= pkg.path.jquery %>/jquery.plugin.pkg.js'
			}

plugin 설치

			$ npm install

minify 실행

			$ grunt min
